number_of_alms

This function returns the number of values stored in each FITS extension in a FITS file containing

Location in HEALPix directory tree: src/f90/mod/fitstools.F90 

FORMAT

var=number_of_alms( filename[, extnum] )

ARGUMENTS

name & dimensionality kind in/out description
       
filename(LEN=filenamelen) CHR IN filename of the FITS-file containing .
extnum I4B OUT number of extensions in the file

EXAMPLE:

print*,number_of_alms('alms.fits')
Prints the number of stored in each extension of the file 'alms.fits'
