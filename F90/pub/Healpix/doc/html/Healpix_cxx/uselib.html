Using Healpix C++ in your own codes

In order to use the Healpix library in your own C++ codes, you have to do two things:

  • #include the appropriate header files in your sources. From this documentation it should be evident which ones you have to include for a specific task.
  • Specify the required libraries when linking your executable. For this release of the software, this should be (order is important!): 
    -lhealpix_cxx -lcxxsupport -lsharp -lcfitsio
