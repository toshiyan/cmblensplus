Changes between releases 3.70 and 3.80

  • Improvement of query_disc routine in inclusive mode,
  • the routines alm2map_spin and map2alm_spin now accept any (integer) spin values , but the scalar routines alm2map and map2alm are still recommended for vanishing spin (),
  • correction of bugs preventing the compilation with versions 10.* of gfortran,
