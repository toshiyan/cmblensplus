vec2ang

Routine to convert the 3D position vector of point into its position angles on the sphere with , , .

Location in HEALPix directory tree: src/f90/mod/pix_tools.F90 

FORMAT

call vec2ang( vector, theta, phi )

ARGUMENTS

name&dimensionality kind in/out description
       
vector(3) DP IN three dimensional cartesian position vector . The north pole is
theta DP OUT colatitude in radians measured southward from north pole (in [0,]).
phi DP OUT longitude in radians measured eastward (in [0, ]).
