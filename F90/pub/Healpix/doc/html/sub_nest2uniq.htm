nest2uniq

This F90 facility turns the parameter (a power of 2) and the pixel index into the Unique ID number . See ”The Unique Identifier scheme” section in ”HEALPix Introduction Document” for more details.

Location in HEALPix directory tree: src/f90/mod/pix_tools.F90 

FORMAT

call nest2uniq( nside, pnest, puniq )

ARGUMENTS

name kind in/out description
       
nside I4B IN The HEALPix parameter.
pnest I4B/I8B IN (NESTED scheme) pixel identification number over the range {0, }.
puniq I4B/I8B OUT The HEALPix Unique pixel identifier.

EXAMPLE:

use healpix_modules
integer(I4B) :: puniq
call nest2uniq(1, 0, puniq)
print*,puniq
returns
4
since the first pixel () at 1 is the pixel with Unique ID number 4.
