Welcome to the cmblensplus documentation¶

This package contains fortran90 wrappers for python to perform CMB lensing/ cosmic bi-refringence/ patchy reionization reconstructions, delensing, and bi-spectrum measurement. The pakage can be downloaded from https://github.com/toshiyan/cmblensplus.

Modules¶

  • basic – Some basic tools
    • delens – analytic calculation for delensing
    • bispec – analytic calculation of bispectrum
    • galaxy – tools for galaxy z distribution
    • cosmofuncs – tools for cosmology related functions
    • flat – cross-check tools for flat-sky normalization
    • wigner_funcs – Wigner 3j symbols
  • curvedsky – Curvedsky analysis tools
    • rec_lens – quadratic lensing reconstruction
    • rec_rot – quadratic rotation reconstruction
    • rec_tau – quadratic tau reconstruction
    • rec_src – quadratic point-soruce reconstruction
    • norm_quad – normalization of quadratic estimator reconstruction
    • norm_imag – normalization of imaginary quadratic estimator reconstruction
    • delens – delensing for simulation
    • bispec – binned reduced bispectrum
    • cninv – Filtering Anisotropies
    • utils – other tools
  • flatsky – Flatsky analysis tools
    • rec_lens – quadratic lensing reconstruction
    • rec_rot – quadratic pol. rot. reconstruction
    • rec_tau – quadratic patchy tau reconstruction
    • rec_src – quadratic point-src reconstruction
    • norm_lens – normalization of quadratic lensing reconstruction
    • norm_rot – normalization of quadratic pol. rot. reconstruction
    • norm_tau – normalization of quadratic patchy tay reconstruction
    • bispec – bispectrum tools
    • utils – other tools
